clone项目到本地

$ cd your_repo_root/repo_name 进入一个目录
$ git clone git@github.com:fxhover/elva.git 克隆本项目到本地
$ git checkout master 切换到master分支

### 安装项目

请先安装好ruby和rails环境 $ bundle install 安装Gemfile中的插件 $ rake db:create 创建数据库 $ rake db:migrate 执行数据库脚本

启动项目

rails s
